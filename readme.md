## 🇧🇷 Português

### 📖 Sobre

Este repositório contém a **versão estática preservada** do site brasileiro **Resident Evil SAC**, um portal dedicado à franquia _Resident Evil_.

O projeto original foi criado por **Felipe Demartini (Evil Shady)** e esteve ativo principalmente entre 2011 e 2013.

O site publicava:

- Notícias
- Artigos editoriais
- Análises
- Vídeos (VideoSAC)
- Conteúdo especial para a comunidade

---

### 🛡️ Preservação Digital

Com o passar dos anos, a instalação original em WordPress tornou-se instável devido a atualizações de servidor, incluindo a migração para PHP 8.

Para evitar perda definitiva do conteúdo histórico, foi criada uma **versão estática completamente independente de banco de dados e PHP**.

Este projeto tem como objetivo:

- 📚 Preservação histórica
- 🌐 Garantir acesso público contínuo
- 🔒 Reduzir dependência tecnológica
- 🧱 Eliminar riscos futuros de incompatibilidade
- 🕰️ Manter viva a memória digital da comunidade

Trata-se de uma iniciativa de **arquivamento digital (digital archiving)**.

---

### ⚙️ Processo de Geração

A versão estática foi criada utilizando:

- WordPress (restaurado temporariamente)
- Plugin Simply Static

Etapas:

1. Restauração da instalação original
2. Correção de incompatibilidades com PHP 8
3. Exportação estática via Simply Static
4. Deploy dos arquivos estáticos

## 🇺🇸 English

### 📖 About

This repository contains a **preserved static version** of the Brazilian fan website **Resident Evil SAC**, dedicated to the _Resident Evil_ franchise.

The original project was created by **Felipe Demartini (Evil Shady)** and was mainly active between 2011 and 2013.

It featured:

- News
- Editorial articles
- Analysis
- Videos (VideoSAC)
- Community-driven content

---

### 🛡️ Digital Preservation

Over time, the original WordPress installation became unstable due to hosting environment updates, including PHP 8 migration.

To prevent permanent data loss, a **fully static version** was generated, removing all database and PHP dependencies.

This project exists to ensure:

- 📚 Historical preservation
- 🌐 Continuous public access
- 🔒 Technological independence
- 🧱 Reduced risk of future incompatibility
- 🕰️ Long-term digital memory preservation

This is fundamentally a **digital archiving initiative**.
